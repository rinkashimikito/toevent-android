# ToEvent

Calendar countdown app for Android. Shows your next event with a live countdown timer.

## Features

- Local calendar, Google Calendar, and Microsoft Outlook support
- Persistent notification with countdown
- Home screen widget
- Floating countdown overlay chip
- Configurable urgency color coding
- Meeting URL detection (Zoom, Meet, Teams, Webex)
- Privacy mode
- Focus mode (filter by calendar)
- Light/Dark/System theme

## Requirements

- Android 10+ (API 29)
- Calendar permission for local events
- Optional: Google/Microsoft account for cloud calendars

## Build

```bash
./gradlew assembleDebug
```
